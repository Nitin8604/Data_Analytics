Project Overview

Heart disease remains one of the leading causes of death worldwide. The increasing prevalence is mainly due to unhealthy lifestyles, poor dietary habits, smoking, alcohol consumption, stress, and lack of physical activity. Early detection and preventive analysis can significantly reduce risks.

This project uses Tableau Public to analyze heart disease datasets and transform raw healthcare data into meaningful insights using dashboards and stories. The objective is to identify key risk factors, understand disease patterns, and support data-driven healthcare decisions.
Problem Statement

Analyzing healthcare datasets manually is difficult due to the large volume of data and multiple influencing factors such as age, BMI, diabetes, smoking habits, and general health conditions.

The main problem addressed in this project is:

How can data visualization help identify key factors contributing to heart disease and stroke risk?

This project aims to solve this problem using SQL data preparation and Tableau dashboards.
Business Requirements

The project focuses on:

Identifying heart disease risk factors

Understanding relationships between lifestyle and disease

Creating interactive dashboards for healthcare analysis

Supporting preventive healthcare decisions

Visualizing patterns between diabetes, stroke, and heart disease
Literature Survey
